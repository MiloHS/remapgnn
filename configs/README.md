# Configs

This directory intentionally contains only the current public configs.

- `v20b_base_a3p0_mink8_geom_v12.json` — current default config for
  `v12_geom_base`.
- `v20b_base_a3p0_mink8.json` — non-geometry baseline config used by the
  current audit/benchmark scripts for older learned comparisons such as `v10b`.

Older lineage, sweep, smoke-test, and rejected-ablation configs are not part of
the public GitHub surface anymore.  Local recoverable copies are under
`archive/` in the development workspace, but `archive/` is ignored by git.
